Free Sample Instructions

This free sample includes basisframeplus4_0.lsp, cube_engine.lsp, and acad3dEIGENVIS.dwt. These files demonstrate the core functionality of the BasisFramePlus module from the LinearAlgebraSuite. Use AutoCAD Student version or the full AutoCAD version — NOT AutoCAD LT.

The template contains all the necessary layers, colors, and visual settings to simulate the full Suite. Point AutoCAD to acad3dEIGENVIS.dwt when creating a new drawing to ensure the sample module loads correctly in acad3D, and with color.

  1. In AutoCAD, type OP and press Enter.
  2. Go to the Files tab.
  3. Expand Support File Search Path.
  4. Click Add.
  5. Click Browse and select the folder where you extracted:
    • basisframeplus4_0.lsp
    • cube_engine.lsp
  6. Click OK to save.

After this, AutoCAD will automatically find and load the sample files when you use APPLOAD and run the BASISFRAMEPLUS4_0 command.

How to Use in AutoCAD

  1. Click Download Free Sample in the navigation bar.
  2. Extract the ZIP file to any folder on your computer.
  3. In AutoCAD, type APPLOAD and load both in this order:
    • cube_engine.lsp
    • basisframeplus4_0.lsp
  4. Run the command: BASISFRAMEPLUS4_0

Sample Input

Use the following standard basis vectors as a test:

e1 = (1, 0, 0)
e2 = (0, 1, 0)
e3 = (0, 0, 1)

This will generate a clean orthonormal frame using the sample module.
